About My Video Series

I am creating a series on binary exploitation and reverse engineering. In this series, we'll take a unique approach by slowing things down to walk through the assembly of a simple piece of C code, step by step, instruction by instruction. We'll use an innovative demo involving an Excel spreadsheet, where each cell represents a memory location, to visualize data movement in and out of CPU registers.

Our journey begins by taking high-level C code and using Godbolt.org to generate its assembly code. We'll then transfer this assembly into our spreadsheet model—our virtual RAM—with dedicated sections to emulate CPU registers like RIP, RSP, and RBP. We'll meticulously walk through each instruction, explaining how function calls are handled, how arguments are passed and returned, and laying the groundwork for understanding more complex concepts in upcoming videos. This series will gradually build up to exploring control structures like loops and delve into buffer overflow attacks, alongside tools like Ghidra, Binary Ninja, and GEF for static and dynamic analysis.
